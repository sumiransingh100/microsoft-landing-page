# Microsoft Homepage Clone

This is a frontend clone of the Microsoft official homepage created using only HTML and CSS.  
The project is built for practicing frontend development, layout structuring, responsiveness, and UI design skills.

## Technologies Used
- HTML5
- CSS3

## Features
- Responsive Design
- Clean UI Layout
- Microsoft-inspired Styling
- Flexbox/Grid Layout

## Purpose
This project was created for frontend practice and improving UI cloning skills.
